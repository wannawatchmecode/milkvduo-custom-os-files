# milkvduo-custom-os-files

This project contains the os files to be added to the milkv
os image to enable ocean and executor services.

These files once added to their corresponding directories of the
milkv os will do the following.

# Enables the following

Update the ip address from the default, randomly generated.

Update host name to a randomly generated hostname ie: milkv-<someRandomString>

Start health services
Start executor service
